Software de Versionamento

É um Software cujo o objetivo é realizar o controle de versões de um código ou documento.
Sendo possível assim, gerenciar as versões de um código ou documento realizadas por um desenvolvedor ou por uma equipe de desenvolvedores.

Tipos de Software de controle de versão.

Centralizado: Possui um repositório central em um servidor com controle de versões.
Depende do servidor.
Softwaresubversion

Distribuído: Possui um repositório local com controle de versões.
Independe do servidor.
Atualizações no repositório central
Softwares mercurial, git

Git

É um software de Versionamento que surgiu para o desenvolvimento do Kernel do Linux, foi projetado e desenvolvido por Linus Torvald em 2005.
Cada diretório de trabalho do Git é repositório completo com um histórico completo e habilidade total de acompanhamento das revisões.
Não dependente de uma rede ou servidor central.

Github

Plataforma de hospedagem de código-fonte e de aquivos com controle de versão.
Surgiu em 2008 e foi comprado pela Microsoft em 2018 por U$7,5 Bilhões.
